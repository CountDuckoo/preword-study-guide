HTML

  • the head element contains information about the webpage
  • the body element represents the visible content shown to the user
  • h1 through h6 designate the level of header
  • always start with !DOCTYPE html
  • wrap all other sections in an html tag that includes the language ("en")
  • meta tags can define character set, description, keywords, author, and starting viewport
  • style defines CSS styles, base defines the base URL, and link connects an external resource to the document
  • img tags include the src (source), alt (alternate text, used for screen-readers or if the image doesn't load), and width and height
  • section tags are semantic elements that define a section that contains thematic content

Git

  • git clone (with a SSH value): to clone a repo from GitHub
  • git status: check what branch we're on, and compare the local status to the branch on GitHub
  • git checkout -b branchname: create a new branch and switch to it
  • git add -A: add all modifications in the current working branch to the staging area
  • git commit -m "comment": create a commit that is a snapshot of the current changes in the staging area
  • git pull origin main: recieve the main (or other) branch from the GitHub repository to make sure you have the latest version
  • git push origin branchname: send the changes that were made locally to a new or existing GitHub branch
